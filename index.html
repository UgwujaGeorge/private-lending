How PrivaLend
works

Every computation — from loan validation to liquidation checks — runs inside Arcium's encrypted MPC environment. No node ever sees your data in plaintext.

01
Encrypt Your Position

Your borrow amount and collateral are encrypted client-side using X25519 keys before being submitted to the blockchain. The values never touch any server in plaintext.

02
MPC Computation

Arcium's cluster of nodes runs the lending logic — collateral ratios, liquidation checks, borrow capacity — using secure Multi-Party Computation over your encrypted data.

03
Encrypted Result

Results are returned encrypted. Only you can decrypt the outcome using your private key. Lenders, liquidators, and node operators never learn your position details.

Architecture

Built with Arcium
MPC

PrivaLend is built on Arcium's encrypted supercomputer — using secure Multi-Party Computation to run financial logic over fully encrypted data.

⚡
Arcis Circuits

Lending logic is written in Arcis — Arcium's Rust-based MPC circuit language. Three circuits handle loan validation, liquidation detection, and capacity computation over encrypted inputs.

🔐
Cerberus Protocol

Arcium's Cerberus MPC protocol provides information-theoretic security. Even a majority of malicious nodes cannot extract or falsify computation results.

⛓
Solana Program

An Anchor-based Solana program queues computations and handles callbacks. Program ID: DVBgkpq...NY63 deployed on devnet.

🌐
MXE Architecture

Multi-Party eXecution Environments (MXEs) isolate each computation. Encrypted state never leaves the MPC cluster — no shared memory, no plaintext exposure.

Encrypted Circuit — validate_loan.arcis
#[instruction]
pub fn validate_loan(
    request: Enc<Shared, LoanRequest>,
) -> Enc<Shared, bool> {
    let r = request.to_arcis();
    // Collateral must be ≥ 150% of borrow — computed encrypted
    let collateral_scaled = r.collateral_amount * 10000u64;
    let borrow_scaled     = r.borrow_amount    * 15000u64;
    let eligible = collateral_scaled >= borrow_scaled;
    // Result returned encrypted — nodes never see plaintext
    request.owner.from_arcis(eligible)
}
